promise()

Description

Code

Demonstration

The promise() method returns a Promise object that represents the completion of all animations on the selected elements. This is useful for ensuring that a series of animations have finished before performing other actions.

$("#animBox").fadeOut(1000) .fadeIn(1000).promise() .done(function() { $(this).css("background-color", "blue"); });
